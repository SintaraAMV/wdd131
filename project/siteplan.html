Website Planning Document

1. Site Name

Proposed name: WDD131 Practical Guide

Why this name: It clearly signals a course-focused site that helps students navigate practical topics and examples in WDD 131.

Optional domain idea: wdd131-practical.example (domain purchase is not required).

2. Site Purpose

This site provides a structured hub for WDD 131 learners to quickly find explanations, examples, and checklists for dynamic web fundamentals. It focuses on practical application: short guides, validated code patterns, and common pitfalls to avoid while building course assignments.

3. Scenarios

  • How can I validate my HTML/CSS and fix the most common errors quickly?
  • Where can I find a proven structure for forms, accessibility, and responsive layouts for course assignments?
